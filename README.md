# Raptor
Raptor' is an simple app to make acceptance tests.

Raptor is powered by:
- NodeJS
- Express
- MongoDb (via Mongoose)
- Passport-local
- Body-parser
- Nodemailer

### HOW TO INSTALL

```bash
$ git clone https://github.com/btronquo/RecetteMaker.git
$ npm-install
$ node app.js
```

Visit http://localhost:3000 & voila !
